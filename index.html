NATO STANAG 4702 1) System Overview

W.I.S.D.O.M™ is a UAV maintenance and operations software (PyQt5) that combines local tools (UI, maintenance workflows, logs, analysis) with optional online services for license validation and a text‑based assistant powered by PLM AI.

Local processing License verification (Render) PLM AI Assistant (text only)
  • Local: maintenance, procedures, UI, vision, logs.
  • Render: machine fingerprint, account ID, license status.
  • PLM AI: only text queries/responses (no ISR/UAV data).
Separation rule: operational UAV/ISR data stays local. Online services are isolated by design.
